Crivo de Eratostenes
====================

Mais informaçãoes sobre o algoritmo: [http://pt.wikipedia.org/wiki/Crivo_de_Erat%C3%B3stenes](http://pt.wikipedia.org/wiki/Crivo_de_Erat%C3%B3stenes)

Como rodar a aplicação
----------------------

Abra um command apontando para a pasta release da raiz do repositório e execute a seguinte linha de comando:

    leonardo_eratosthenes.exe N T

N é o número limite e T é o numero de threads (ambos são inteiros).

Ex:

    leonardo_eratosthenes.exe 20 4
